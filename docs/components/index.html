      Содержание страницы: Группы компонентов

      Компоненты BootstrapVue и плагины для групп компонентов.

      Alert — Предоставляйте контекстные сообщения обратной связи для типичных действий пользователя с помощью нескольких доступных и гибких предупреждающих сообщений. Aspect — Компонент `<b-aspect>` может использоваться для поддержания минимального адаптивного соотношения сторон содержимого. v2.9.0Avatar — Аватары обычно используются для отображения профиля пользователя в виде изображения, иконки или короткого текста. v2.8.0Badge — Небольшой адаптивный тег для добавления контекста практически к любому контенту. Breadcrumb — Укажите местоположение текущей страницы в навигационной иерархии. Разделители автоматически добавляются в CSS через ::before и content. Button — Используйте специальный компонент Bootstrap b-button для действий в формах, диалоговых окнах и т. д. Включает поддержку нескольких контекстных вариаций, размеров, состояний и т. д. Button Group — Сгруппируйте несколько кнопок в одну строку с помощью <b-button-group>. Button Toolbar — Сгруппируйте серию <b-button-group> и/или <b-input-group> вместе в одной строке с дополнительной навигацией с клавиатуры. Calendar — Пользовательский виджет календаря BootstrapVue для выбора дат и управления другими компонентами, полностью доступен WAI-ARIA (a11y) и поддерживает интернационализацию (i18n). v2.5.0Card — Карточка - это гибкий и расширяемый контейнер содержимого. Он включает в себя варианты верхних и нижних колонтитулов, широкий спектр контента, контекстные цвета фона и мощные параметры отображения. Carousel — Компонент <b-carousel> представляет собой слайд-шоу для циклического просмотра серии контента, созданного с помощью преобразований CSS 3D. Он работает с серией изображений, текста или пользовательской разметки. Collapse — Легко переключайте видимость контента на своих страницах. Включает поддержку для изготовления аккордеонов. Dropdown — Выпадающие списки - это переключаемые контекстные наложения для отображения списков ссылок и действий в формате раскрывающегося меню. Embed — Создавайте адаптивные встраивания видео или слайд-шоу на основе ширины родительского элемента, создавая внутреннее соотношение, которое масштабируется на любом устройстве. Form — Компонент формы и вспомогательные компоненты формы, которые дополнительно поддерживают встроенные стили формы и состояния проверки. Form Checkbox — Пользовательский ввод флажка и группа флажков для замены ввода флажка по умолчанию в браузере, построенного на основе семантической и доступной разметки. Опционально поддерживает стиль переключателя. Form Datepicker — Элемент управления формой ввода настраиваемого средства выбора даты BootstrapVue, который обеспечивает полную доступность WAI-ARIA (a11y) и поддерживает интернационализацию (i18n). v2.5.0Form File — Настраиваемый, кроссбраузерный, элемент управления вводом файлов, поддерживающий загрузку одного файла, нескольких файлов и каталогов. Form Group — Легко добавляйте некоторую структуру к формам. Его цель состоит в том, чтобы объединить элементы управления формы с легендой или меткой и предоставить текст справки и недопустимый/действительный текст обратной связи, а также визуальную (цветную) обратную связь с контекстным состоянием. Form Input — Создавайте различные типы входных данных, такие как: текст, пароль, номер, URL-адрес, адрес электронной почты, поиск, диапазон, дата и т. д. Form Radio — Кроссбраузерные радиовводы и радиогруппы с использованием настраиваемого радиоввода Bootstrap для замены радиоввода браузера по умолчанию. Form Rating — BootstrapVue настраиваемый элемент управления формой рейтинга в звездах для ввода оценок или агрегированного отображения. v2.12.0Form Select — Пользовательский компонент <select> с использованием настраиваемых стилей для разных браузеров. При желании сгенерировать записи <option> на основе массива, массива объектов. Form Spinbutton — Компонент ввода настраиваемой числовой спин-кнопки BootstrapVue, обеспечивающий доступность WAI-ARIA (a11y) и интернационализацию (i18n). v2.5.0Form Tags — Легкий пользовательский элемент управления формой ввода с тегами, с параметрами для настраиваемого рендеринга интерфейса, обнаружения повторяющихся тегов и дополнительной проверки тегов. v2.2.0Form Textarea — Создавайте многострочные текстовые поля ввода с поддержкой автоматического изменения размера, минимального и максимального количества строк и состояний контекстной проверки. Form Timepicker — Элемент управления формы ввода настраиваемого средства выбора времени BootstrapVue, который обеспечивает полную доступность WAI-ARIA (a11y) и поддерживает интернационализацию (i18n). v2.6.0Image — Создавайте адаптивные изображения, при желании добавляя к ним легкие стили - и все это с помощью свойства. Поддержка закругленных изображений, стили миниатюр, выравнивания и даже возможность создавать пустые изображения с дополнительным сплошным цветом фона и ленивыми загруженными изображениями. Input Group — Легко расширяйте элементы управления формой, добавляя текст, кнопки или группы кнопок по обе стороны от текстовых полей ввода Jumbotron — Легкий и гибкий компонент, который при желании может расширять всю область просмотра для демонстрации ключевых маркетинговых сообщений на вашем сайте. Layout and Grid System — Используйте мощную сетку flexbox для мобильных устройств (с помощью компонентов <b-container>, <b-row>, <b-form-row> и <b-col>) для создания макетов всех форм и размеров благодаря двенадцати система столбцов, пять адаптивных уровней по умолчанию, переменные и примеси CSS Sass и десятки предопределенных классов. Link — Используйте настраиваемый компонент BootstrapVue <b-link> для создания стандартной ссылки <a> или <router-link>. <b-link> поддерживает отключенное состояние и распространение события клика. List Group — Группы списков — это гибкий и мощный компонент для отображения серии содержимого. Элементы группы списка могут быть изменены для поддержки практически любого содержимого внутри. Их также можно использовать в качестве навигации с помощью различных свойств. Media — Медиа-объект помогает создавать сложные и повторяющиеся компоненты, в которых некоторые медиафайлы располагаются рядом с содержимым, которое не обертывается вокруг указанного медиафайла. Modal — Модальные окна оптимизированы, но гибкие диалоговые подсказки основаны на JavaScript и CSS. Они поддерживают ряд вариантов использования, от уведомлений пользователей до полностью настраиваемого контента, и содержат несколько полезных подкомпонентов, размеров, вариантов, доступности и многого другого. Nav — Компоненты навигации, использующие общую разметку и стили, от базового класса <b-nav> до активных и отключенных состояний. Меняйте свойства модификатора, чтобы переключаться между каждым стилем. Navbar — Компонент <b-navbar> — это адаптивная оболочка, которая размещает брендинг, навигацию и другие элементы в сжатом заголовке. Его легко расширять благодаря компоненту <b-collapse>. Overlay — Компонент b-overlay используется для визуального скрытия определенного элемента или компонента и его содержимого. Он сигнализирует пользователю об изменении состояния элемента или компонента и может использоваться для создания загрузчиков, предупреждений/предупреждений и многого другого. v2.7.0Pagination — Кнопки быстрой первой, предыдущей, следующей, последней и страницы для управления разбиением на страницы другого компонента (например, <b-table> или списков). Pagination Nav — Быстрые кнопки первая, предыдущая, следующая, последняя и страница для разбиения на страницы на основе навигации, поддерживающие обычные ссылки или ссылки маршрутизатора. Popover — The Popover feature provides a tooltip-like behavior, can be easily applied to any interactive element via the <b-popover> component or v-b-popover directive. Progress — A custom progress component for displaying simple or complex progress bars, featuring support for horizontally stacked bars, animated backgrounds, and text labels. Sidebar — The `<b-sidebar>` component creates a fixed viewport, left or right, sliding popout drawer. v2.10.0Skeleton — The <b-skeleton> component can be used to scaffold a loading state, while your data is loading. v2.17.0Spinner — The <b-spinner> component can be used to show the loading state in your projects. They're rendered only with basic HTML and CSS as a lightweight Vue functional component. Table — For displaying tabular data. <b-table> supports pagination, filtering, sorting, custom rendering, events, and asynchronous data. For simple display of tabular data without all the fancy features, BootstrapVue also provides lightweight alternative components <b-table-lite> and <b-table-simple>. Tabs — Create a widget of tabbable panes of local content. The tabs component is built upon navs and cards internally, and provides full keyboard navigation control of the tabs. Time — A custom time input widget that can be used to control other components v2.6.0Toast — Push notifications to your visitors with a toast, a lightweight and easily customizable alert message. Tooltip — Easily add tooltips to elements or components via the <b-tooltip> component or v-b-tooltip directive.
