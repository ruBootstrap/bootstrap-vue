Пользовательские темы и информационные панели

С темами и информационными панелями, созданными нашими партнерами, вы можете создавать привлекательные приложения и страницы — все используют BootstrapVue! Следующие элементы были подготовлены командой BootstrapVue.

BootstrapVue Argon Dashboard PRO

BootstrapVue Argon Dashboard PRO — это совершенно новый продукт, основанный на нашей новейшей перестроенной с нуля структуре фреймворка, который призван сделать наши продукты более интуитивно понятными, более адаптивными и, разумеется, намного более простыми в настройке. Позвольте Argon удивить вас своими замечательными функциями и инструментами для сборки и вывести ваш проект на совершенно новый уровень

Категория: Admin & Dashboard Предоставлено: Creative Tim

Получить dashboard Стоимость: $89.00*

BootstrapVue Argon Dashboard состоит из более чем 100 отдельных компонентов, что дает вам свободу выбора и комбинирования. Все компоненты могут иметь разные цвета, которые можно легко изменить с помощью файлов SASS.

Gull — это современная панель администратора Vue.js нового поколения. Он многофункциональный, отзывчивый и построен на основе Vue CLI, Vuex, Vue Router и BootstrapVue. Если вы хотите создать панель администратора Vue.js, лучше всего подойдет Gull.

Vuexy — это готовая к работе, тщательно продуманная обширная панель администратора. В основном он работает на BootstrapVue и API композиции Vue.js. Панель управления использует API с аутентификацией JWT и контролем доступа на основе возможностей.

Примечания:

  • Цены указаны в долларах США. Цены подвержены изменениям. Текущие цены смотрите на веб-сайте вендора/провайдера.

Вы представитель темы?

Если вы заинтересованы в том, чтобы стать партнером и разместить свою тему или информационную панель на этой странице, ваш продукт должен соответствовать следующим требованиям:

  • Он должен быть основан на компонентах BootstrapVue (или расширять их).
  • Должен быть совместим с BootstrapVue, чтобы пользователи могли при желании включать собственные компоненты BootstrapVue.
  • Следует избегать необходимости в файлах JavaScript jQuery или Bootstrap v4 (за исключением включенных сторонних компонентов, если это необходимо).
  • Должен предоставить исходные файлы/переменные SCSS/SASS.
  • Следует продвигать, что он основан на (или совместим с) BootstrapVue.
  • Пользовательские компоненты, предоставляемые темой, должны быть доступны через WAI-ARIA. Любые ограничения WAI-ARIA должны быть отмечены в документации к теме.
